# Azelea test app

### What is this repo?

This repo is a tester for the Azelea PHP Framework



### How do I run this?

When the repo is installed, type ```composer install``` into your terminal

Make an VirtualHost in XAMPP, or any other webhoster, to load ```/public/index.php```
